# Меню недели — Android (Jetpack Compose, Clean Architecture)

Приложение для составления меню на неделю с управлением продуктами
("холодильник"), рецептами и списком покупок.

## Стек
- Kotlin + Jetpack Compose, Material 3 (включая Dynamic Color на Android 12+)
- Чистая архитектура: `domain` (модели, интерфейсы репозиториев, use cases) →
  `data` (Room, реализации репозиториев, Hilt-модули) → `presentation`
  (ViewModel + Compose-экраны)
- Room — офлайн-хранение данных
- Hilt — внедрение зависимостей
- Navigation Compose — навигация между разделами
- Coil — загрузка фото рецептов

## Функционал
- **Холодильник**: добавление/изменение/удаление продукта (название, иконка,
  единица измерения, количество).
- **Рецепты**: создание своего рецепта — название, фото (выбор из галереи),
  ингредиенты (можно выбрать из холодильника или добавить произвольный),
  пошаговые инструкции приготовления.
- **Меню на неделю**: вручную назначить рецепт на день недели и приём пищи
  (завтрак/обед/ужин/перекус).
- **Список покупок**: добавление ингредиентов из холодильника, отметка
  купленного, удаление.

## Как открыть
1. Открыть папку проекта в Android Studio (Koala или новее).
2. Дождаться синхронизации Gradle (зависимости подтянутся автоматически).
3. Запустить на эмуляторе/устройстве с Android 7.0+ (minSdk 24).
